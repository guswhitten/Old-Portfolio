Equipment Specs:

  • Texas Instruments' Code Composer Studio IDE: Download

  • Part 1 constraints:

  • define a range of acceptable temps using a max and min
  • define an array of 5 temps that your systems collects every second and average them
  • if average falls within temp range, green LED on
  • if average falls outside temp range, red LED on

    Part 2 constraints:

  • when push button is pressed 4 times: red LED turns on, green LED will be off
  • when push button pressed 2 more times: green LED turns on, red LED turns off
  • this pattern repeats indefinitely

    Part 3 constraints:

  • at the beginning of the program, the green LED will be on
  • include a counter variable that counts number of times the push button is pressed
  • when push button is pressed 5x, or any multiple of 5, green LED turns on
  • the button pressing routine should be defined in an Interrupt Service Routine (ISR)
